Static website (HTML/CSS/JS) motivating Bulgarians toward healthy living. 5 pages, linear flow, dark theme. Python scraper for weekly grocery offers. All content in Bulgarian.

## Coding rules

**General**
- Prefer proven libraries over hand-rolled code. Use `Fuse.js` for fuzzy search, `BeautifulSoup`/`requests` for HTML extraction, `pdfplumber` for PDFs.
- Don't add features, comments, error handling, or abstractions beyond what the task requires.
- Don't create new files unless necessary вАФ extend existing ones.
- No backwards-compatibility shims. If something is unused, delete it.
- When a big problem takes multiple sessions to solve, document what was tried in a `NOTES.md` next to the relevant code.

**HTML/CSS**
- One CSS file (`css/style.css`) вАФ no inline styles, no new CSS files.
- Reuse existing components: `.stat-card`, `.accordion-item`, `.info-box`, `.btn-green`, etc.
- Color modifiers `.green`, `.red`, `.amber`, `.purple` work on most components вАФ use them instead of custom colors.
- All new text must have `data-i18n` or `data-i18n-html` attribute pointing to a key in `js/i18n.js`.
- Current copy workflow is BG-only. Do not add or update EN keys during normal content edits.
- `js/i18n.en.parked.js` exists only to preserve past EN work. Do not load it, merge it back, or spend tokens reading/editing it unless the task is explicitly about EN translations or re-enabling multilingual support.
- Responsive breakpoint is 480px вАФ test mobile after layout changes.

**JavaScript**
- No frameworks вАФ vanilla JS only. No new script files without a strong reason.
- DOM queries go inside `DOMContentLoaded` or the relevant `init*()` function.
- Tracker data uses localStorage key `nutrilife-tracker-YYYY-MM-DD`.
- Don't duplicate offer filtering logic вАФ extend `renderOffers()` in `offers.js`.

**Python (scraper)**
- Read `scraper/NOTES.md` before changing scraper code.
- `data/offers.json` and `data/all_products.json` are generated вАФ never edit manually.
- After any scraper run, always sync with `python sync_offers.py`.
- Scraper uses `SOURCE_PRIORITY`: structured DOM (3) > brochure listing (2). New sources must fit this ranking.
- Don't add new per-store scrapers to `scraper.py` directly вАФ add to `scraper/store_scrapers.py` and import.
- Temporary/debug data files go in `data/` with a clear name prefix. Delete them when done вАФ don't commit them.

## Scraper workflow
